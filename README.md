# 🤖 手撕 Agent

> 从零开始，手把手带你拆解 Agent 技术栈。

## 📖 项目简介

本项目是 Agent 的学习与实践仓库，通过**动手造轮子**的方式深入理解 Agent 的核心原理。从最简单的 ReAct 模式开始，逐步探索 Function Calling、MCP、A2A 协议、主流框架到多 Agent 协作系统。
