# 🤖 Build Agent from Scratch - 智能点餐助手

## 📖 简介

本目录包含一个从零实现的 Agent 示例，使用 ReAct 范式（推理+行动），通过正则表达式解析工具调用。

## 🚀 快速开始

```bash
# 设置环境变量
export TENCENT_API_KEY="你的API密钥"

# 运行 Agent
python3 order_agent.py
```

## 📁 文件说明

| 文件             | 说明                         |
| ---------------- | ---------------------------- |
| `order_agent.py` | 智能点餐 Agent（正则解析版） |

---

## 🔧 可用工具

### 1️⃣ ask_menu_price - 查询菜品价格

```
查询单个菜品的价格

参数：
  - item_name: 菜品名称

示例：
  [Call: ask_menu_price: 咖啡]
  [Call: ask_menu_price: 汉堡]
```

### 2️⃣ calculate - 计算总价

```
计算数学表达式

参数：
  - expression: 数学表达式

示例：
  [Call: calculate: 25*2 + 8*1]
  [Call: calculate: 15 + 12 + 8]
```

## 💬 使用示例

```
👤 用户: 我要2份汉堡和1杯可乐

==================================================
第 1 轮对话
==================================================

Thought: 查询汉堡价格[Call: ask_menu_price: 汉堡]
Action: ask_menu_price(汉堡)
Observation: 汉堡的价格是25元

==================================================
第 2 轮对话
==================================================

Thought: 查询可乐价格[Call: ask_menu_price: 可乐]
Action: ask_menu_price(可乐)
Observation: 可乐的价格是8元

==================================================
第 3 轮对话
==================================================

Thought: 计算总价[Call: calculate: 25*2 + 8*1]
Action: calculate(25*2 + 8*1)
Observation: 58元

==================================================
第 4 轮对话
==================================================

Thought: 得到总价58元，输出答案
Answer: 您的订单：汉堡x2=50元，可乐x1=8元，总计58元

✅ 点餐完成!
```
